## Cara Mengupload

1. Letakkan file foto profil di: `assets/images/profile.jpg`
2. Letakkan logo Unsri di: `assets/images/unsri_logo.png`
3. Restart aplikasi Streamlit jika sudah berjalan

## Catatan

- Jika file tidak ditemukan, aplikasi akan menampilkan placeholder
- Gunakan foto dengan pencahayaan baik untuk hasil terbaik
